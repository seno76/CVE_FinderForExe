## Детали по платформам

### Windows

**Все функции поддерживаются:**
- Сканирование файлов и папок
- Сканирование реестра (все установленные программы)
- Program Files и Program Files (x86)
- Множественные диски
- Комбинированное сканирование

**Запуск:**

```bash
python apps/interactive_scanner.py
```

### Linux

**Поддерживаемые функции:**
- Сканирование файлов и папок
- Системные папки: /usr/bin, /usr/local/bin, /opt
- Пользовательские папки
- Полное системное сканирование

**Не поддерживается:**
- Сканирование реестра (специфично для Windows)
- Сканирование всех дисков (используй опцию сканирования папки вручную)

**Запуск:**

```bash
python apps/interactive_scanner.py
# или
python3 apps/interactive_scanner.py
```

**Пример сканирования /opt:**

```
Выберите опцию (0-10): 3
Введите путь папки: /opt
✅ Добавлена: opt
Введите следующий путь (или Enter для завершения): 
```

### macOS

**Поддерживаемые функции:**
- Сканирование файлов и папок
- Системные папки: /usr/bin, /usr/local/bin, /opt
- Пользовательские папки
- Полное системное сканирование

**Запуск:**

```bash
python3 apps/interactive_scanner.py
```

## Требования по ОС

### Windows

- Python 3.8+
- Права администратора (для сканирования реестра)
- 100+ MB свободного места

### Linux

- Python 3.8+
- Права доступа к папкам (sudo может потребоваться для некоторых директорий)
- 100+ MB свободного места

### macOS

- Python 3.8+
- Права доступа к папкам
- 100+ MB свободного места

## Рекомендации

### Для лучших результатов на Windows

1. Используйте **опцию 9** (реестр) для получения всех установленных программ
2. Это автоматически найдёт программы на всех дисках
3. Это быстрее и полнее, чем сканирование папок

### Для лучших результатов на Linux

1. Используйте **опцию 8** (полное системное сканирование)
2. Если нужны права администратора: `sudo python3 apps/interactive_scanner.py`
3. Для сканирования пользовательских программ используйте опцию 2 или 3

### На любой ОС

- JSON отчёты совместимы со всеми платформами
- HTML отчёты открываются в любом браузере
- Отчёты можно использовать для автоматизации и интеграции с другими системами

---

## Известные ограничения

- Сканирование реестра работает только на Windows (специфично для реестра Windows)
- На Linux некоторые программы могут не иметь очевидной версии в имени файла
- macOS имеет специфичную структуру приложений (./app/Contents/MacOS/), может потребоваться дополнительная настройка
